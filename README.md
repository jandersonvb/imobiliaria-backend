# Imobiliária Backend

API do marketplace e CRM imobiliário, construída com NestJS, Prisma e PostgreSQL.

## Escopo inicial

- autenticação e usuários
- imobiliárias e membros
- corretores
- imóveis e mídias
- leads e funil comercial
- visitas e propostas
- favoritos, chat e planos

## Stack

- NestJS
- TypeScript
- Prisma ORM
- PostgreSQL
- JWT
- Swagger

## Execução local

```bash
npm install
cp .env.example .env
npm run prisma:generate
npm run start:dev
```
